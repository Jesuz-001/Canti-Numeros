 Que hace el programa?
 
solicita al usuario ingresar n cantidad de números enteros. Luego analiza y cuenta cuántos números son pares, impares, mayores a 10, positivos y negativos. Al final muestra todos los resultados en pantalla.

Que datos recibe de entrada?

1. La cantidad de números que el usuario desea ingresar.
2. Cada número entero que el usuario digite.

3. Que datos procesa?

El programa procesa cada número ingresado para clasificarlo según:
- Si es par o impar: numero % 2 == 0
- Si es mayor a 10: numero > 10
- Si es positivo o negativo: numero > 0 o numero < 0
Y va aumentando contadores para cada categoría.

Que datos imprime?

Al finalizar imprime en pantalla:
- Cantidad de números pares
- Cantidad de números impares
- Cantidad de números mayores a 10
- Cantidad de números positivos
- Cantidad de números negativos
